# Starkiller

## Starkiller

Starkiller is a Frontend for [Powershell Empire](https://github.com/BC-SECURITY/Empire/). It is a web application written in VueJS. If you'd like to contribute please follow the [Contribution guide](https://github.com/BC-SECURITY/Empire/blob/main/CONTRIBUTING.md). If you'd like to request a feature or report a bug, please follow the [Issue template](https://github.com/BC-SECURITY/Empire/blob/main/.github/ISSUE_TEMPLATE.md).

## Getting Started

As of Empire 5.0 and Starkiller 2.0, you no longer need to install Starkiller or build it from source. It is prepackaged in Empire as a submodule and served via Empire's API.
